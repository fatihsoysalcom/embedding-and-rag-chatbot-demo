# Embedding and RAG Chatbot Demo

This Python script demonstrates a simplified Retrieval-Augmented Generation (RAG) system. It simulates embedding text documents and a user's question, then retrieves the most relevant document to 'augment' a generated answer, mimicking how advanced chatbots access external knowledge.

## Language

`python`

## How to Run

Run this script using Python: `python rag_chatbot.py`
